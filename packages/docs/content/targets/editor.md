# Editors

`target.editor` should hold shared editor and workspace conventions.

## First Targets

- VS Code `settings.json`
- `.vscode/extensions.json`

## Why It Matters

These files are copied between repositories constantly.

Typed presets would make it much easier to share:

- formatting defaults
- TypeScript language settings
- extension recommendations
- workspace ergonomics
